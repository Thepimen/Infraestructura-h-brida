# Infraestructura-h-brida
Este proyecto implementa un SOC con Wazuh sobre una infraestructura híbrida de Docker y VMs. Monitoriza entornos vulnerables como Windows 11 y DVWA , enfrentándolos a ataques desde BlackArch y NixOS. Ofrece visibilidad de $360^{\circ}$ mediante análisis de logs, FIM y detección XDR.
